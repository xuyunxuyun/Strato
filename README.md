## 📝 极简操作指南

1. **唤醒**：打开DeepSeek，开启新会话
2. **投喂**：先上传`prompt.md`，再上传`characters.md`，无需提示词，无需深度思考，打开联网搜索，直接发送
3. **召唤**：发送玩法咒语，并根据超级团队的执行结果，选择继续或提出新要求

## 📝 **【扩展】自定义人设：打造你的专属梦之队**

不想用三国人物？想让你心中的偶像、虚构的角色，甚至你的朋友同事组团为你打工？完全可以。

**操作方式**：按下方模板编写「人设文件」（.md格式），代替原人设文件，你的专属团队即刻出道。

### 📁 人设文件模板（参考「汉末废柴回收站」格式）

```markdown
# 超级团队组建附件：［你的团队名称］人物设定

## 核心决策团队

### ［角色1姓名］：**［职位，如：首席执行官］**

1. **身份**：［一句话定位］
2. **核心能力**：［核心标签，2-3个］
3. **性格**：
    - ［性格特点1］：［具体描述］
    - ［性格特点2］：［具体描述］
    - ［性格特点3］：［具体描述］
4. **核心关系**：
    - ［关系对象］：［关系描述］
    - ［关系对象］：［关系描述］
5. **惯用语**：
    - “［经典台词1］”
    - “［经典台词2］”
6. **习惯动作**：
    - ［习惯性动作描述］
7. **核心信念**：
    - “［人生信条1］”
    - “［人生信条2］”

### ［角色2姓名］：**［职位］**

……（以此类推，覆盖所有团队）
```

### ✍️ **编写小贴士**

| 要点                | 说明                                                         |
| ------------------- | ------------------------------------------------------------ |
| **人物关系**        | 让角色之间有互动、有羁绊，团队才有灵魂（师徒/宿敌/挚友/父子） |
| **惯用语+习惯动作** | 这是让人物“活起来”的关键，一句话就能让人记住                 |
| **字数建议**        | 每个人物200-500字为宜，不用太长，但要传神                    |
| **核心信念**        | 人物的精神内核，决定ta在关键时刻的选择                       |
